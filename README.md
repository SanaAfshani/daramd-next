# MVP فارسی (Next.js + Tailwind CSS)

این پروژه یک نمونه‌ی سبک (MVP) از رابط کاربری موبایل به زبان فارسی/RTL است که با **Next.js 14** و **Tailwind CSS** پیاده‌سازی شده و از طرح ارسالی شما الهام گرفته شده است.

## اجرای پروژه

```bash
npm install
npm run dev
# سپس در مرورگر به آدرس http://localhost:3000 بروید
```

> توجه: اگر Node.js نسخهٔ جدید دارید، بسته‌ها با نسخه‌های نزدیک نصب می‌شوند. در صورت نیاز می‌توانید نسخه‌ها را در `package.json` به‌روزرسانی/پین کنید.

## ساختار پوشه‌ها

```
src/
  app/
    page.tsx           # صفحهٔ اصلی (خانه)
    layout.tsx         # چیدمان عمومی با فونت فارسی و RTL
    globals.css        # استایل‌های Tailwind
    funds/page.tsx     # صفحات نمونهٔ تب‌ها
    messages/page.tsx
    account/page.tsx
  components/          # اجزای UI (TopBar، کارت تبلیغی، اکشن‌ها، تراکنش‌ها، ناوبری پایین)
  lib/
    data.ts            # داده‌های نمونه
```

## نکات طراحی

- **RTL** کامل با `dir="rtl"` و فونت فارسی **Vazirmatn**
- پالت رنگی نزدیک به طرح: سبز برند (`brand`) و نارنجی اکسن (`accent`)
- کارت پرومو با اسلاید خودکار، اکشن‌های سریع، کارت ویدیو و لیست تراکنش‌های نمونه
- ناوبری پایین با هایلایت تب فعلی

## توسعه‌ی بعدی

- اتصال به API واقعی تراکنش‌ها
- صفحهٔ جزئیات «حساب درآمد ویژه»
- وضعیت‌های Skeleton/Loading
- دسترسی‌پذیری و تست‌های E2E
